Detailed Description

The Block Class.

This class describes how to create a block object All the Block variables need to be set when the class is instantiated. it is possible to get the left,right,top,bottom values of the Ball it is possible to get the number of X, and Y ( column / Rows ) of the Ball it is possible to get the Total number of blocks ( x * y )
